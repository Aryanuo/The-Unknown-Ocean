# The Unknown Ocean

A self-contained browser prototype for an infinite, living ocean expedition.

Open `index.html` in a browser to start. The app stores expedition progress, mapped regions, named species, and photos locally in the browser.

Included systems:

- Cinematic surface-to-underwater entry
- WebGL 3D ocean scene with procedural biomes, terrain, creatures, and coordinates
- Third-person diver hero controlled with keyboard, pointer, and dive/ascend buttons
- Animated 3D fish with tail motion, banking, wandering, avoidance, and behavior variation
- Deterministic species discoveries with naming rights
- Research log and community-style progress stats
- Daily global event rotation
- Ambient generated audio after expedition start
- Screenshot capture from the ocean canvas
